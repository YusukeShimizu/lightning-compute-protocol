LCP (Lightning Compute Protocol) は、Lightning 上で「小さな計算ジョブ」に対して支払いを行うためのアプリケーション層プロトコルです。
直接の Lightning ピア接続上で、BOLT #1 のカスタムメッセージを使用します。
